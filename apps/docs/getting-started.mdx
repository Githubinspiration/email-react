## 1. Install dependencies

Install one of the components from your command line.

<CodeGroup>

```sh npm
npm install @react-email/button @react-email/render -E
```

```sh yarn
yarn add @react-email/button @react-email/render -E
```

</CodeGroup>

## 2. Create an email

Add a button component to your email template. Include styles where needed.

```jsx email.tsx
import { Button } from '@react-email/button';

const Email = () => {
  return (
    <Button href="https://example.com" style={{ color: '#61dafb' }}>
      Click me
    </Button>
  );
};
```

## 3. Send your email

Import the email template you just built, convert into a HTML string, and use your preferred email service provider to send it.

```jsx index.tsx
import { render } from '@react-email/render';
import { Email } from './Email';

const html = render(<Email />);
```
